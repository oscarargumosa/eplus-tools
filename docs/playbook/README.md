# The FUN-DESIGN™ Playbook

### *Designing Better European Projects*

**El documento de referencia de EU Funding Studio** · Borrador editorial v0.1

---

> No vendemos IA. Vendemos **Better Projects**.

Este libro es el activo intelectual fundacional de **EU Funding Studio**: no un manual
técnico ni una memoria, sino el *playbook* que explica **por qué** existe FUN-DESIGN™,
**cómo** se diseña un proyecto europeo ganador y **hacia dónde** va el ecosistema.

Está pensado para poder usarse, dentro de unos años, para presentar la empresa a
inversores, formar consultores, registrar la metodología, entregar a clientes premium
y posicionar a EU Funding School como referente en diseño de proyectos europeos.

---

## La metodología en una imagen

**FUN-DESIGN™** son nueve fases —el acróstico F·U·N·D·E·S·I·G·N—, cada una con su
herramienta y su entregable, que se encadenan: el entregable de una fase es el cimiento de
la siguiente.

```
   L   FUN-DESIGN™                          Herramienta     Entregable
   ─   ──────────────────────────────       ────────────    ─────────────────────────
   F   Find the Opportunity                 🔎 EUFinder     Funding Opportunity
   U   Unlock and Share the Project Vision  🔓 EUUnlocker   Project Vision + Publication
   N   Nail Down the Activities             🏗 EUDesigner   Project Architecture
   D   Deliver the Concept Note             📨 EUConnector  Concept Note + Invitations
   E   Engage the Consortium                🔗 EULinker     Consortium Strategy
   S   Show the European Added Value        🔬 EUResearcher Research + Added Value
   I   Integrate the Delivery Plan          🛡 EUAuditor    Audit-Ready Delivery Plan
   G   Generate the Proposal                ✍️ EUWriter     High-Quality Proposal
   N   Nurture Future Opportunities         🏆 EUEvaluator  Evaluation + Improvement
```

> EU Funding Studio no es un generador de propuestas. Es una **consultoría europea de
> financiación digital**, donde cada herramienta de IA actúa como un experto especializado
> dentro de un equipo coordinado.
